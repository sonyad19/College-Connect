# College Connect

College Connect is a web app designed to help college students connect with each other, find clubs and events, share resources, and build a stronger campus community.

## Features (Planned)

- Student profiles (basic info, major, interests)
- Events board (campus events, club meetings, hackathons)
- Discussion board (ask questions, share tips, study groups)
- Club & organization directory
- Search (by major, interests, or event type)

## Tech Stack (Starter Version)

- HTML, CSS, JavaScript (front-end only for now)
